# 权限管理

根据系统设置的安全规则或者安全策略，既定用户可以访问而且只能访问自己被系统管理员授权的目录。
权利不同可以了解的数据内容不同，保证数据隐私，避免数据泄密；职责不同所需要页面不同，保证操作效率，避免页面干扰；技术水平不同可以操作的功能不同，保证系统安全，避免操作风险。
